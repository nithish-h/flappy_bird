# Flappy Bird AI (NEAT)

Flappy Bird AI created with NEAT (Neuroevolution of Augmenting Topologies).

## Live Demo

[View Live Site Here]()

If it appears zoomed in, just press `Ctrl -`

Some shortcuts:

- `q` - 1x Speed
- `w` - 2x Speed
- `e` - 5x Speed
- `r` - 10x Speed
- `t` - 30x Speed
- `y` - 650x Speed

- `v` - Toggle bird vision visualization
